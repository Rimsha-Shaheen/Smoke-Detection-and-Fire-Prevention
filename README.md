# Smoke Detection and Fire Prevention

An IoT-based fire prevention system using Cisco Packet Tracer and virtual sensors.

## Features
- Real-time smoke detection
- Automated fire alert system
- Virtual IoT sensors integration
- Simulated using Cisco Packet Tracer

## Files
- `SmokeDetection.pkt` – Packet Tracer project file
- `screenshots/` – Images of the network and sensor setup
- `docs/` – Additional documentation and project ppt file

## Tools Used
- Cisco Packet Tracer
- IoT virtual sensors
